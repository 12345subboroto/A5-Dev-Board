Question-3: What is event delegation in the DOM?

Event delegation is a technique where instead of adding event listeners to multiple child elements, you add a single event listener to their parent. It works by taking advantage of event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and propagates upwards to its parent elements.
